# 杰理官方文档说明

我们提供的杰理官方资料有：
1. JL7016G Datasheet
2. JL7016G TWS耳机标准原理图
3. API 参考
<img src={jl_docs} alt="firmware" width="400" style={{ display: 'block', margin: '0 auto' }} />
因为官方只提供了中文版本的开法文档，所以不熟悉中文的开发者可能会面临一些挑战。



为了帮助开发者快速查找所需内容。我会简单介绍一下上述资料。

## 1 JL7016G Datasheet

文件位置：`./Firmware-JL701N/doc/datasheet/JL7016G Datasheet V1.3.pdf`
JL7016G Datasheet V1.3 主要是介绍 杰理 JL7016G 芯片的详细规格，是给硬件工程师或嵌入式开发者使用的技术手册


## 2 JL7016G TWS耳机标准原理图 

文件位置：`./Firmware-JL701N/doc/datasheet/JL7016G Datasheet V1.3.pdf`
这份原理图文档是为使用 JL7016G 芯片开发高集成度 TWS 蓝牙耳机产品（具备 ANC + ENC + 空间音效）提供的完整硬件设计参考，包含接口连接、滤波/防护、电源布局、焊接建议等，确保产品在功能性、可靠性和可制造性方面符合要求。

## 3 API 参考

文件位置：`./Firmware-JL701N/doc/html/`
这是一份非常详细的API说明文档。对于想在Buddie 基础上进行二次开发的用户，这份文档会带来非常大的帮助。
这份文档包含但不局限于如下几个方面：
1. SDK技术框架
2. 操作系统的API
3. 文件系统的API
4. 外设驱动的API
5. 蓝牙协议栈开发说明

需要注意的是，文档的版本和当前的SDK版本并不完全匹配，可能会存在以下问题：
1. SDK中存在无法在文档中查询到的函数接口
2. 文档中的部分API在SDK中不支持
